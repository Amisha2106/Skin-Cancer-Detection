# Skin-Cancer-Detection

A Deep Learning project to detect skin cancer from images using CNNs, originally developed in Google Colab.

## Overview
This project uses a Convolutional Neural Network (CNN) to classify skin lesions and assist in the early detection of skin cancer.

## Features
* Image preprocessing and augmentation.
* Deep Learning model using TensorFlow/Keras.
* Performance evaluation with accuracy and loss curves.

## Getting Started
1. **Open in Colab:** Click the badge above to run the notebook directly in your browser.
2. **Local Run:** * Clone the repository.
   * Install dependencies: `pip install -r requirements.txt`.
   * Open `SKIN_CANCER_DETECTION_.ipynb` in Jupyter Notebook.
